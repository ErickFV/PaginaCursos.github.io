React

React te permite construir interfaces de usuario a partir de piezas individuales llamadas componentes. Crea tus propios componentes de React como Thumbnail, LikeButton, y Video. Luego combínalos para formar pantallas, páginas y aplicaciones.

Contenido Del Curso
1 Introducción a React: Se comienza con una introducción a React, su filosofía y su enfoque en la construcción de interfaces de usuario interactivas. También se explica cómo configurar el entorno de desarrollo de React en tu computadora.
2 Componentes en React: Se enseñan los conceptos fundamentales de los componentes en React, cómo crearlos y utilizarlos para construir interfaces modulares y reutilizables. Se explica la diferencia entre componentes funcionales y componentes de clase, y se muestran las mejores prácticas para su uso.
3 Estado y propiedades: Se explora el manejo del estado y las propiedades en React. Se enseña cómo manejar el estado de los componentes, actualizarlo y propagarlo a través de la jerarquía de componentes utilizando las propiedades. También se muestra cómo utilizar el estado y las propiedades para lograr una comunicación eficiente entre componentes.
4 Ciclo de vida de los componentes: Se analiza el ciclo de vida de los componentes en React, desde la creación hasta la eliminación. Se explican los métodos del ciclo de vida y cómo utilizarlos para realizar tareas específicas en cada etapa, como la inicialización de datos, la manipulación de eventos y la limpieza de recursos.
5 Enrutamiento y navegación: Se introduce el enrutamiento en aplicaciones React utilizando bibliotecas populares como React Router. Se enseña cómo configurar rutas, navegar entre páginas y utilizar parámetros dinámicos en las URL.
6 Gestión de eventos: Se muestra cómo manejar eventos en React, tanto eventos del navegador como eventos personalizados. Se explora cómo vincular eventos a los componentes y cómo manejar las interacciones del usuario de manera efectiva.
7 Gestión del estado global: Se aborda la gestión del estado global en aplicaciones React mediante bibliotecas como Redux o Context API. Se enseña cómo centralizar y compartir el estado entre múltiples componentes y cómo actualizarlo de manera consistente.
8 Integración con APIs y servicios: Se muestra cómo utilizar JavaScript y las bibliotecas adecuadas para realizar solicitudes a APIs y servicios web. Se cubren temas como la obtención de datos, el manejo de respuestas y la actualización dinámica de la interfaz de usuario.
9 Estilos y CSS en React: Se exploran las diferentes formas de aplicar estilos y CSS en aplicaciones React. Se enseña cómo utilizar CSS en línea, hojas de estilos externas y bibliotecas populares de estilos como Styled Components o CSS Modules.
10 Despliegue y optimización de aplicaciones: Se presentan las mejores prácticas para el despliegue de aplicaciones React en entornos de producción. Se discuten temas como la optimización del rendimiento, la división de código, la carga diferida y la implementación en servicios de alojamiento.
